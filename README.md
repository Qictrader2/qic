# New Stories

Stories in this folder were extracted from **NEW_FEATURES.md** (Registration, Profile & Affiliate Fixes) and converted to the same format as the other stories in `stories/`.

- **Source:** `NEW_FEATURES.md` (QicTrader Platform — User Stories, 06/02/2026)
- **Format:** One file per story; each has title, **As a / I want to / so that**, Acceptance Criteria, and Testing Instructions (placeholder).
- **Naming:** `{ID}-{slug}.md` (e.g. `REG-001-task-bar-icon-hover-effect.md`).

**FEATURES.md** is the full feature inventory (API endpoints + frontend features) and lives at the repo root; it is a catalog rather than user stories, so it is not split into individual story files here.
